# 🏏 Hardik Himanshu Pandya Analysis Dashboard | Power BI Project

## 📌 Project Introduction

The Hardik Himanshu Pandya Analysis Dashboard is a Power BI project designed to analyze the cricket performance statistics of Indian all-rounder Hardik Pandya across different match formats and venues.

This dashboard transforms raw cricket statistics into interactive visual insights that help users understand batting performance, captaincy records, match-winning contributions, strike rate trends, and overall career achievements.

---

# 🎯 Project Objective

The main goals of this project are:

* To analyze Hardik Pandya’s overall cricket performance
* To monitor batting and captaincy statistics
* To compare match performances across formats and venues
* To identify winning contributions and performance trends
* To create an interactive sports analytics dashboard using Power BI
